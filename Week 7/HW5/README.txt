//CAMPUS TASK BOARD API

== PROJECT DESCRIPTION ==
A Spring Boot REST API that allows users to create, read, update, and delete tasks which all allows a working backend server with input validation.


== HOW TO RUN THE APPLICATION == 
1. Open project file in IntelliJ IDE
2. Run CampusTaskboardApplication.java
3. Go to http://localhost:8080 in browser


== API ENDPOINTS DOCUMENTATION == 
GET /api/tasks  
POST /api/tasks  
GET /api/tasks/{}  
PUT /api/tasks/{}  
DELETE /api/tasks/{}
